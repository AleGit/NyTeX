\begin{definition}[\CNF]
A (first order) \coloremph{literal} \( L \) is either
an atom \( A \) or the negation \( (\lnot A) \) of an atom.
We usually abbreviate \( \lnot(s\mEQ t) \) with \( s\mNE t \).
Further we call atoms \coloremph{positive} literals
and negated atoms \coloremph{negative} literals.
%
We define the \coloremph{complement} \( L^\mcc \) of a literal
as follows:
%
\[
\MDEFINE{L^\mcc}{rll}{
	\lnot A
	&\text{if }L = A\text
	&\text{where }A\text{ is an atom}
	\\
	A
	&\text{if }L = \lnot A
	&\text{where }A\text{ is an atom}
}
\]
%
A (first order) \coloremph{clause} \( \mcC = L_1\lor\ldots\lor L_n \)  is a possibly empty multiset of literals.
A finite \coloremph{set of clauses} \( S=\{ \mcC_1,\ldots,\mcC_n \} \)Â is in \coloremph{clausal normal form}. 
\end{definition}

\begin{remark}
	First order terms, atoms, literals, clauses, formulae, sets of clauses
	are first order expressions.
	We call a first order expression without variables \coloremph{ground},
	i.e.\ we build ground first order expressions over an empty set of variables.
	Obviously, ground first order formulae cannot contain quantifiers
	that depend on variables.
\end{remark}
